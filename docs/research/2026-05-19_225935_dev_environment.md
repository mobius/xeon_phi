# Xeon Phi 7120P 开发环境研究

> 时间: 2026-05-19 22:59
> 状态: 研究完成，待安装

## 1. 架构概览

| 维度 | 详情 |
|------|------|
| CPU 架构 | k1om (Knights Corner), x86-64 子集 |
| 指令集 | 无 SSE/AVX, 专用 512-bit ZMM (IMCI) |
| 卡内 OS | Linux 2.6.38.8+mpss3.8.6, BusyBox |
| 卡内 libc | glibc 2.14.1 |
| 卡内内存 | 16 GB GDDR5 |

## 2. 开发模型

### Native 模式（推荐）
主机交叉编译 → scp 上传 → ssh 卡内执行

### Offload 模式
编译时 #pragma offload → 运行时 COI 自动卸载

### 异构 MPI
主机 MPI + 卡内 MPI，通过 SCIF 通信

## 3. 可用工具链（tar 包中）

| RPM | 内容 |
|-----|------|
| `intel-composerxe-compat-k1om` | GCC 4.7 交叉编译器 (x86_64-k1om-linux-gcc/g++) |
| `mpss-sdk-k1om` | k1om sysroot (glibc 2.14.1, BusyBox, 所有库) |
| `mpss-offload-dev` | Offload 开发头文件 |
| `mpss-offload` | Offload 运行时库 |

## 4. 快速测试

安装后：

```bash
/usr/linux-k1om-4.7/bin/x86_64-k1om-linux-gcc \
    --sysroot=/opt/mpss/3.8.6/sysroots/k1om-mpss-linux \
    -o hello.mic hello.c

scp hello.mic mic0:/tmp/
ssh mic0 /tmp/hello.mic
```
