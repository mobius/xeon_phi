## 2. 物理兼容性 — ✅ 完全可行

ESC4000G4 **就是为装多张双宽加速卡设计的**。4 张卡（3 VE + 1 Phi）在物理上完全没问题：

- 机箱空间: 2U 高度支持全高全长双宽卡 ✅
- 插槽数量: 4-6 个双宽插槽可用 ✅
- 当前只装了 1 张 VE，还有 3-5 个空槽 ✅

---

## 3. 散热评估 — ⚠️ 勉强可行，但 7120P 是瓶颈

### 3.1 ESC4000G4 散热系统

| 参数 | 数值 |
|------|------|
| 风扇数量 | 8 个前置风扇模块 |
| 风扇尺寸 | 40mm x 56mm (或 60mm) |
| 风扇转速 | 10,000 - 15,000 RPM |
| 单个风量 | ~20-30 CFM |
| **总风量** | **~160-240 CFM** |
| 噪音 | ~70-80 dB(A) 满载 |

### 3.2 热量预算

| 组件 | 热量 |
|------|------|
| 3x NEC VE 1.0 | ~900W |
| 1x Xeon Phi 7120P | ~300W |
| 2x Xeon Gold 6252 | ~300W |
| 其他 | ~100W |
| **总计** | **~1600W** |

### 3.3 2U 服务器散热能力

| 场景 | 散热能力 | 结论 |
|------|---------|------|
| 优秀风道 + 风扇满载 | ~1500-2000W | ✅ 刚好够 |
| 一般风道 + 正常转速 | ~1000-1200W | ❌ 不够 |
| 前面板堵塞 / 灰尘 | ~800-1000W | ❌ 严重不足 |

**结论**: ESC4000G4 的散热系统在**理想条件下**可以 handle 1600W，但余量很小。

### 3.4 7120P 被动散热的致命问题

```
进风口 ──→ [Slot 1] ──→ [Slot 2] ──→ [Slot 3] ──→ [Slot 4] ──→ 出风口
          25°C空气      35°C空气       45°C空气       55°C空气
```

7120P 没有自带风扇，**完全依赖机箱风道**。如果放在 Slot 3 或 Slot 4：
- 进风已被前面 VE 卡加热到 40-50°C
- 300W 被动散热在 50°C 进风条件下几乎不可能
- 会立即 thermal throttle（降频到 20-30%）

**唯一可行位置: Slot 1（最靠近进风口）**

| 位置 | 进风温度估计 | 7120P 可行性 |
|------|-------------|-------------|
| Slot 1 (最左) | 25°C | ✅ 勉强可行 |
| Slot 2 | 35°C | ⚠️ 可能降频 |
| Slot 3 | 45°C | ❌ 严重降频 |
| Slot 4 (最右) | 55°C | ❌ 不可行 |

## 4. 推荐插槽布局

### 4.1 最优布局

```
前面板进风口
     │
     ▼
┌─────────────────────────────────────────────┐
│  Slot 1        Slot 2        Slot 3         │
│  ┌─────┐      ┌─────┐      ┌─────┐         │
│  │7120P│  →   │ VE  │  →   │ VE  │         │
│  │(被动)│      │ #1  │      │ #2  │         │
│  └─────┘      └─────┘      └─────┘         │
│      ↑                          ↑           │
│   冷空气                      热空气         │
│                                             │
│  Slot 4                                     │
│  ┌─────┐                                    │
│  │ VE  │ ← 现有卡可保留或迁移                │
│  │ #0  │                                    │
│  └─────┘                                    │
└─────────────────────────────────────────────┘
     │
     ▼
  后面板出风
```

### 4.2 布局说明

| Slot | 设备 | 原因 |
|------|------|------|
| Slot 1 | **7120P** | 最先接触冷空气，被动散热唯一可行位置 |
| Slot 2 | VE #1 (新) | 有主动风扇，对进风温度不敏感 |
| Slot 3 | VE #2 (新) | 有主动风扇 |
| Slot 4 | VE #0 (现有) | 可保留，或移到 Slot 2/3 |

**注意**: 现有 VE 卡在 5e:00.0，需要确认 5e:00.0 对应哪个物理 Slot。

## 7. 最终结论

### 7.1 ESC4000G4 装 3 VE + 7120P — 可行但有条件

| 条件 | 状态 | 说明 |
|------|------|------|
| 物理空间 | ✅ | 2U GPU 服务器设计就是干这个的 |
| PCIe 插槽 | ✅ | 4-6 个双宽槽足够 |
| 散热 | ⚠️ | 刚好够，但 7120P 必须放 Slot 1 |
| 电源 | ❓ | 需确认当前 PSU 功率 |
| 软件驱动 | ⚠️ | 需要 jjkeijser 社区补丁 |

### 7.2 必须满足的条件

1. **7120P 必须装在 Slot 1**（最靠近进风口）
2. **电源 ≥ 2000W**（如果是 1600W 必须升级）
3. **前置风扇必须全速运转**（BIOS 设置 Performance Mode）
4. **机房环境温度 ≤ 25°C**
5. **前置面板不能堵塞**（硬盘位满不影响，但防尘网要清洁）

### 7.3 7120P 预期性能

即使在 Slot 1 的最优位置：

| 场景 | 预期性能 |
|------|---------|
| 理想散热 (25°C 进风) | 标称的 70-80% |
| 一般散热 (30°C 进风) | 标称的 50-60% |
| 满载长时间运行 | 可能 thermal throttle 到 30-40% |

** realistic FP64**: ~0.8-1.0 TFLOPS（标称 1.2 TFLOPS）

### 7.4 决策建议

```
你有 7120P 了吗?
├── 还没有
│   └── ❌ 不要买。ESC4000G4 装 4 张 VE 1.0 才是最佳方案
│
└── 已有 7120P
    ├── 电源 < 2000W?
    │   └── ⚠️ 升级电源后再考虑
    ├── 机房无空调 (温度 > 30°C)?
    │   └── ❌ 7120P 会严重降频，不建议装
    ├── 愿意接受 7120P 可能只有 50-70% 性能?
    │   ├── 否 → 单独找个机箱装 7120P
    │   └── 是 → 可以装，放 Slot 1
    └── END
```
